# Volt Athletic

Volt Athletic is a modern and responsive sports equipment landing page built with HTML5 and CSS3. It focuses on clean design, responsive layouts, and a user-friendly experience.

## Live Demo

https://cheshtajain-02.github.io/volt-athletic/

## Features

- Responsive Layout
- Sticky Navigation Bar
- Hero Section
- Product Showcase
- About Section
- Customer Testimonials
- Contact Form
- Modern Footer
- Smooth Hover Effects

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries
- Font Awesome
